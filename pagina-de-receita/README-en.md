# Recipe Page

- [Read this README in Portuguese](README-pt.md)
- [View the live project](https://erikaestudar.github.io/course-project/pagina-de-receita/index.html)

This project is a simple styled recipe page created as part of a front-end challenge. The page showcases a coffee cupcake with whipped cream recipe, including an image, ingredients list, and preparation instructions.

## Technologies used

- HTML5
- CSS3
- Google Fonts (`Alice`)

## Included files

- `index.html`: page structure with semantic content.
- `styles/index.css`: styling and page layout.
- `assets/images/`: images used in the project, such as the cupcake photo and heart icon.

## Project details

- Designed for clear reading and visual presentation.
- Uses semantic sections and article elements.
- Includes a featured image to highlight the recipe.
- Shows preparation time, servings, and difficulty level.

## How to view

1. Visit the live site: [Recipe Page](https://erikaestudar.github.io/course-project/pagina-de-receita/index.html).
2. Or open `index.html` locally in your browser.
3. Alternatively, use a simple local server such as Live Server in VS Code.
